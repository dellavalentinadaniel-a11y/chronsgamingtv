# UI Design Excellence Skill

This skill defines the visual standards for the **ChronsGamingtv** project to ensure a premium, modern, and high-impact user interface.

## Core Design Principles

1. **Rich Aesthetics**:
    - Use vibrant, curated color palettes (no generic colors).
    - Implement sleek dark modes as the primary experience.
    - Leverage **Glassmorphism**: Subtle backgrounds with `backdrop-filter: blur()`.

2. **Typography**:
    - Use modern fonts like `Inter`, `Roboto`, or `Outfit`.
    - Maintain a clear typographic hierarchy (single `<h1>`, logical headings).

3. **Motion and Interaction**:
    - Use smooth gradients and transitions.
    - Implement micro-animations for hover states and transitions using `motion`.

4. **Visual Polish**:
    - Avoid placeholders; use `generate_image` for realistic assets.
    - Ensure responsive layouts for all device types.

## Design Tokens

- **Primary**: Sleek deep blues or purples.
- **Accent**: Vibrant teals or neon highlights.
- **Surfaces**: Semi-transparent dark layers with borders.

## SEO Best Practices

- Proper Title Tags and Meta Descriptions on every page.
- Semantic HTML (header, main, footer, section).
- Unique IDs for all interactive elements to facilitate testing.
